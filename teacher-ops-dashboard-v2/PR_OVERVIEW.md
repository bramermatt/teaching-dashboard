# Teacher Ops Dashboard v2 PR Bundle

This file exists to explicitly bundle the new `teacher-ops-dashboard-v2` scaffold into a single PR-ready change set.

Included in this bundle:
- React + TypeScript + Vite scaffold
- TailwindCSS setup and dark ops theme tokens
- Typed dashboard domain model
- LocalStorage persistence hook
- Reusable panel component
- Starter mission-control dashboard layout and panels

Use this folder as the isolated v2 track while preserving the existing production dashboard.
